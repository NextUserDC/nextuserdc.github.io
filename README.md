# 🌐 NextUserDC

> [nextuser.lat](https://nextuser.lat) — Portfolio de proyectos web

## 📋 Descripción

Sitio web estático alojado en GitHub Pages que alberga una colección de proyectos web personales. Desarrollado con HTML, CSS y JavaScript puro (sin frameworks ni build tools).

## 🚀 Proyectos

### 🎮 MCAccounts

Buscador de cuentas de Minecraft no premium con base de datos indexada por prefijo (3 caracteres), service worker para soporte offline e infinite scroll para renderizado eficiente de grandes volúmenes de resultados.

**Tecnologías:** Service Worker, JSON indexado, `requestAnimationFrame` para scroll, debounce en búsqueda.

### 🕹️ GameFinder

Motor de búsqueda de ofertas de videojuegos en tiendas legales usando la API de CheapShark, con pestaña adicional para sitios alternativos.

**Tecnologías:** Fetch API, DOM manipulation, Google Fonts (Outfit).

### 💕 Camila

Página web personal dedicada con cuenta regresiva, mini-juego Wordle como mecanismo de desbloqueo, álbum de fotos, sección de sentimientos y lista de Spotify embebida.

**Tecnologías:** Lucide Icons, Google Fonts (Cinzel, Metal Mania, Quicksand), API de WhatsApp (CallMeBot).

### 💻 Sistemas Operativos

Directorio de descargas directas de 19+ sistemas operativos y 3 herramientas USB (balenaEtcher, Ventoy, Rufus) con búsqueda en tiempo real y logos SVG personalizados.

**Tecnologías:** SVG custom, CSS Grid, filtro de búsqueda en JavaScript.

### 🍽️ La Mesa 58

Sitio web de restaurante venezolano con menú completo (comestibles, bebestibles, postres, sopas), sección de cultura venezolana y diseño responsive con animaciones scroll-driven.

**Tecnologías:** Google Fonts (Rubik), CSS Scroll-Driven Animations, `loading="lazy"`.

## ⚙️ Tecnologías

- 🌐 HTML5 / CSS3 / JavaScript (vanilla)
- 📦 GitHub Pages con dominio personalizado
- 🔧 Service Workers (MCAccounts)
- 🎨 CSS Grid, Flexbox, Glassmorphism
- 🖼️ Imágenes optimizadas (WebP, JPEG comprimido)
